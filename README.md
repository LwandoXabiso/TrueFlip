A full-stack **social media profile marketplace platform** built using the **PERN stack (PostgreSQL, Express, React, Node.js)**.

Users can list, buy, and sell verified social media profiles through a secure marketplace with admin verification, subscription billing, and automated background processing.

## 🚀 Features

### 👤 Users
- Secure authentication via **Clerk**
- Create and manage listings
- Browse marketplace profiles
- Purchase verified social media accounts
- Receive credentials via email

### 🛠 Admin
- Verify and approve listings
- Update profile credentials before publishing
- Manage users and orders

### 💳 Payments & Subscriptions
- Clerk Billing integration
- Premium subscription plans
- Free trial support

### 📬 Automation
- Email notifications for orders & listings
- Background job processing using **Inngest**
- Clerk webhook handling
